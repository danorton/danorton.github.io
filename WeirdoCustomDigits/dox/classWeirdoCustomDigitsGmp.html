Detailed Description

Implementation of WeirdoCustomDigits using the gmp extension for arithmetic.

Requirements

  • gmp extension ( See http://php.net/gmp )

Limitations

  • None more than WeirdoCustomDigits.

The "internally represented" format for values of this subclass is the internal value that gmp uses, e.g. the value returned by gmp_init().
