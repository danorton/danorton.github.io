WeirdoCustomDigits Documentation

This module provides the abstract WeirdoCustomDigits PHP class and implementation subclasses, which perform radix conversion on numbers with arbitrary digits. Details are in the documentation for the WeirdoCustomDigits module.

This source is available at GitHub. To fetch it using git:

git clone 'https://github.com/danorton/WeirdoCustomDigits.git'
