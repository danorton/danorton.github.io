Detailed Description

Abstract class for converting custom numbers with arbitrary radixes and digit characters.

Requirements

  • PHP 5.3 or later
  • If using multi-byte or UTF-8 digits, PCRE UTF-8 support is required.

Limitations

  • Only non-negative integers ("whole numbers") are supported.

Class constructor.

Parameters
[in]string | array$digitsset of case-sensitive-distinct characters that form the digits of a custom number. The first digit is zero, &c. If an array is passed, each element represents a single digit. (Numbers with multi-CHARACTER digits can be rendered, but not parsed, although multi-BYTE Unicode characters are just fine). Default is WeirdoCustomDigits::DIGITS_70.
[in]int$radixnumber of digits (i.e. the radix). Default is the number of digits provided in $digits.
[in]bool$use_uctrue if to allow Unicode characters in the custom number's set of digits, else allow ASCII only. Default is true.

Example:

// Manipulate base-65 numbers that have the digits [0-9a-zA-Z_.*].
$wrdx = new WeirdoCustomDigitInt(WeirdoCustomDigits::DIGITS_70,65);
